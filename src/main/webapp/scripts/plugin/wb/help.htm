1.什么是微信CRM？
微信CRM是一款发起联盟合作的通讯录软件。通过发起通讯录，可以帮您梳理圈子的电话、职业，供需信息等形成联盟。帮助行业圈子或微信群友认识彼此，深化关系，加强合作。

具体控制做法

微信CRM屏蔽了所有网页端的访问，所有的通讯录只能在手机端上才能录入读取。我们对每个通讯录都进行的加密处理，来保障数据的安全。同时我们共有3种方式方式，保护群通讯录隐私，安全：

1、需要群主审核可见，需要通过群主或者管理员审核才可以查看通讯录。

2、建立群通讯录的时候，群主设立通讯录密码，只有输入密码的人才能看到。

即使通过了上面2关群主的审核，我们还有第三步个人设置

3、个人也可以对自己名片中手机号码进行隐藏设置，设置为对群友可见，或者指对群主可见。

2. 关注后，点击通讯录选择发起通讯录，或者回复1，就可以发起通讯录了

7.如何设置管理员，删除不必要人员，群成员如何交换名片？

群管理是个组织行为，如何设置他人一起来管理呢?管理人员必须先录入名片，录入后发起人可以点开这个人的通讯录，里面有设置管理员，删除成员，与其交换名片等按钮进行设置。

具体设置如下图

8.群主发起后，再点进链接发现无法管理或者设置群成员？

这是微信授权过期的问题，你只要找到公众号(微信CRM)的通讯录，从这里进去后，再授权一次，就可以了。

9.分享时候没有出现分享按钮问题？

网络原因导致页面部分代码没有加载完成，只要刷新一下页面即可。
